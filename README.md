# Village Sim — City Builder & Tactical Battles

A Unity 2D strategy game combining village/city-building progression with tactical battle gameplay.

The project is inspired by Anno-style city builders and Fire Emblem-style combat. The player builds and manages a growing village, assigns villagers to production roles, collects resources, upgrades buildings, and prepares units for tactical battles.

## Features

### Village / City Simulation

- Grid-based building placement
- Roads and walkable paths
- Resource production and consumption
- Villager assignment system
- Building construction and upgrades
- Production buildings with worker slots
- Inventory and shop menus
- Camera movement and zoom
- UI panels for building, editing, shop, missions, and overview menus

### Tactical Battle System

- Separate battle flow from the city simulation
- Unit deployment zones
- Grid-based unit positioning
- Villager/unit stats such as attack, defense, range, mobility, evasion, and critical chance
- Ranged and melee unit behavior
- Combat inspired by tactical RPGs

## About

Village Sim is a personal Unity project focused on combining two gameplay loops:

1. Building and managing a village economy
2. Using villagers/units in tactical grid-based battles

The goal is to create a strategy game where the city-building side directly supports the battle side.
